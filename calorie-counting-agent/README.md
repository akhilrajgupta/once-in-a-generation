# Calorie Counting Agent

A modern web application for tracking daily calorie and fuel intake.

## Key Features
*   **Frontend**: Built with **React**, **Vite**, and **TypeScript** for a fast, modern experience.
*   **UI**: Uses **Shadcn UI** for a polished, accessible, and responsive interface.
*   **Visualization**: Integrated charts (via **Recharts**) for tracking progress over time.
*   **Backend**: Connected to **Supabase** for secure data persistence and authentication.

## Setup
1.  **Install**: Run `npm install` to install dependencies.
2.  **Environment**: Ensure you have your Supabase credentials configured in `.env`.
3.  **Run**: Start the development server with `npm run dev`.
