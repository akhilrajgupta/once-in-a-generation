# AI Travel Assistant

An intelligent assistant that plans trips by finding flights, hotels, and activities.

## Key Features
*   **Multi-Service Search**: Integrates with **SerpAPI** for Flights, Hotels, and Activities.
*   **Calendar & Email**: Automatically creates Google Calendar events and sends email summaries.
*   **MCP Architecture**: Uses Model Context Protocol for tool modularity.

## Setup
1.  **Import**: Import the workflow JSON.
2.  **Credentials**: Configure **OpenAI**, **SerpAPI**, **Gmail**, and **Google Calendar**.
3.  **Endpoints**: Update MCP Client nodes to point to your local or hosted MCP server endpoints.
